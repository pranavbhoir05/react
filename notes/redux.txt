1. What Redux is
A state management library used with React (but works with any JS app).
It manages large, complex, shared state in a predictable way.

ğŸ“Œ 3. Redux core rules
Redux has:

One store (global state)
Reducers (pure functions to update state)
Actions (describe the update)
Dispatch (send an action)
Selectors (read specific data)

Everything is predictable and centralized.
